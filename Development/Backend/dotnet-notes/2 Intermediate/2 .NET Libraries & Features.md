# .NET Libraries & Features (Intermediate Notes)

## 1. File I/O, Streams

* **File I/O**: Reading/writing files using `System.IO`.

  ```csharp
  File.WriteAllText("test.txt", "Hello World");
  string content = File.ReadAllText("test.txt");
  ```
* **Streams**: Handle data as a sequence of bytes.

  * `FileStream`, `MemoryStream`, `StreamReader`, `StreamWriter`.
  * Used for efficient reading/writing of large files.

---

## 2. Memory Management & Garbage Collection

* **Automatic memory management** handled by the CLR.
* **Garbage Collector (GC)**:

  * Frees unused objects from memory.
  * Uses generations (Gen 0, 1, 2) for efficiency.
* **Dispose & Finalize**:

  * Implement `IDisposable` and `Dispose()` for releasing unmanaged resources.
  * Use `using` statement for automatic cleanup.

  ```csharp
  using (var reader = new StreamReader("file.txt"))
  {
      string line = reader.ReadLine();
  }
  ```

---

## 3. Dependency Injection (DI)

* **DI**: Design pattern for loose coupling.
* Instead of creating dependencies directly, inject them via constructor.

  ```csharp
  class Service
  {
      public void Execute() => Console.WriteLine("Running Service");
  }

  class App
  {
      private readonly Service _service;
      public App(Service service) => _service = service;
      public void Run() => _service.Execute();
  }
  ```
* **Built-in DI container** available in ASP.NET Core.

---

## 4. NuGet Packages

* **NuGet**: Package manager for .NET libraries.
* Provides reusable components like Entity Framework, AutoMapper, Serilog.
* Install via **Visual Studio Package Manager** or CLI:

  ```bash
  dotnet add package Newtonsoft.Json
  ```
* Keeps projects modular and up-to-date.
